# Permissions Matrix

## CLIENT
Può:
- visualizzare prodotti
- visualizzare categorie
- registrarsi e fare login
- creare una recensione
- visualizzare le proprie recensioni
- visualizzare le recensioni approvate dei prodotti
- confrontare prodotti usando l'AI
- segnalare recensioni

Non può:
- creare, modificare o eliminare prodotti
- visualizzare tutte le recensioni
- approvare o nascondere recensioni
- gestire utenti
- accedere alle analisi statistiche admin
- visualizzare la dashboard di sistema

## MODERATOR
Può:
- visualizzare tutte le recensioni
- approvare recensioni
- nascondere recensioni
- eliminare recensioni inappropriate
- gestire segnalazioni

Non può:
- creare, modificare o eliminare prodotti
- gestire utenti
- accedere alle funzioni amministrative complete

## ADMIN
Può:
- fare tutto ciò che può fare il moderatore
- creare, modificare ed eliminare prodotti
- gestire categorie
- gestire utenti
- accedere alle analisi AI
- accedere alla dashboard amministrativa
